糟糕的一天

经过了n个周天和半个多月的暑假在电脑前不停的敲键盘，从静态到动态，从coding到github，从.tk到.xyz再到.top，从Jekyll到Hexo再到Wordpress，最终还是回到了Hexo，虽然有点难以置信但事实就是这样，期间电脑被我一键win7 64—>win8.1 32，本来是win10的但是居然还失败了，不知是我不会用这软件的缘故还是电脑已经老了，毕竟电脑之前被我拍到系统盘损坏，嗷嗷嗷最后还是从包里拿出了U盘老老实实的重装了（哭~）。
好了，今天是大喜…啊不对，开始？，好像还是不对，应该是运行的日子吧。
总结一下我敲了多少次code…
首先是在git上cd到Hexo文件夹，然后是$npm install -g hexo-cli，接着打了一行$hexo init，$npm install生成了文件，敲了一行$hexo g，$hexo s看到了曙光，最后是$hexo d.
我不造上面的有没有敲错，总之，敲就对了。
好了，那么你可能会问:“为什么标题要写成‘糟糕的一天’，今天不是应该高兴才对吗？”
我会这么跟你说:“高兴你个头啊，走了半天回到起点，这不是白费劲吗。的确，是应该高兴，但是我一点也高兴不起来，因为，键盘膜给我敲出了个洞。又得花钱了（哭）”
不敲了，lol首胜还没打呢，待会又把键盘膜敲坏了…
